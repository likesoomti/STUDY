# 기본 위젯들



## 1) 기본 위젯들

버튼 또는 글자가 보이는 텍스트 뷰가 존재한다. 

기본적으로 사용되는 것들이다. 

그정도 선에서 많이 쓰이는 위젯을 봅시다!



### 텍스트 뷰

기본중의 기본

버튼도 텍스트 뷰를 상속해서 만들었다!

#### 주요한 속성

- text : 텍스트에 보이는 문자열 설정
- textColor : 텍스트 색상
- textSize : 텍스트 크기 설정
- typeFace: 폰트 설정
- maxLines: 몇 줄로 표현할 것인지. 줄이 길면 짤린다
- lineSpacingExtra : 줄 간격을 떨어트린다
