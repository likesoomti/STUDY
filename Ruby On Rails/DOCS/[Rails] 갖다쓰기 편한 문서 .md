## 로그인 로그아웃 헬퍼

```erb
<% unless(user_signed_in?) %>
  <a href="/users/sign_in">로그인 페이지</a>
<% else %>
  <div class="user-name"><%=current_user.name%>님</div>
  <%= link_to "로그아웃", destroy_user_session_path, :method => :delete %>
<% end %>
```
