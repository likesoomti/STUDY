## study1. 

- 정적: 한번 연결되어서 계속 자동적으로 바뀐다. 
- 동적: 한번 호출. 
- html붙으면 정적. 안붙으면 동적이라고 생각하면 된다. 
- temp.jade는 js 문법을 쓴다. 이걸 그냥 동적 파일로 생각하면 된다. 
- .html파일은 정적으로 계속 가져오는 것. 

- 동기 : 내가 처리하는거 ( 양이 많으면 시간이 걸린다. ) 순서대로 하는 것들

- 비동기 : 남에게 넘기는거 : 콜백 : 다했다고 알려줌. 

- sync는 콜백함수가 달린다.
